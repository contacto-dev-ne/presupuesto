¿Cómo funciona?

La aplicación descomprime dinámicamente en caliente cada archivo datos_[año].json.gz utilizando la librería Pako. Puedes configurar una URL base personalizada para cargar tus datos desde cualquier repositorio de GitHub.

Usa ./ para rutas locales relativas en tu hosting GitHub Pages, o ingresa la URL de tu raw de GitHub.
